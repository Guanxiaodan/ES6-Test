## 对ES6的理解：
每年的6月，会发布一版标准，用年份标记，所以就是见到的ECMAScript 2015,ECMAScrpt 2016之类的。
ECMA2015开始就是ES6了，ECMA2016可以看为ES6.1，ECMA2017可以看作为ES6.2

ES6是在ES5基础上在语法和API上都有较大幅度的升级。

ES5是对ES3的小幅升级，ES3就是我们刚开始接触JavaScript时候学的那些语法和API。
